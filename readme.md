# О сборке
Базовая сборка React + TypeScrypt + ESLint (Airbnb) с помощью Vite.

## install
```
npm i
```

## dev server
```
npm run dev
```

## production
```
npm run build
```
сборка в папке dist

## run Eslint check
```
npm run lint
```

## run Prettier fix files
```
npm run fix
```

## run Jest test(s)
```
npm run test
```
